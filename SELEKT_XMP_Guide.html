SELEKTで判定した写真のカラーラベルをXMPファイルとして書き出し、現像ソフトに反映する方法を解説します。

使い方
1
セレクト画面で写真を判定する
各写真に OK / 保留 / NG のステータスをつけます。
グループ一括操作やキーボードショートカット（O / H / N）も使えます。
2
「XMP書き出し」をクリック
画面右上（モバイルは下部）の XMP書き出し → ボタンをクリックします。
確認ダイアログに OK / 保留 / NG の各枚数が表示されます。
3
ZIPファイルをダウンロード
「📁 XMPをダウンロード」をクリックすると、判定済みの写真に対応するXMPファイルがZIPにまとめてダウンロードされます。
未判定の写真にはXMPは生成されません。
4
写真と同じフォルダにXMPを配置
ダウンロードしたZIPを展開し、元の写真ファイルと同じフォルダにXMPファイルを入れます。
📁 撮影フォルダ/
├── DSC_0001.jpg
├── DSC_0001.xmp ← ここに配置
├── DSC_0002.jpg
├── DSC_0002.xmp ← ここに配置
├── DSC_0003.ARW
├── DSC_0003.xmp ← ここに配置
└── ...
5
現像ソフトで読み込み
Lightroom や Capture One でフォルダを読み込むと、カラーラベルが自動的に反映されます。
⚠
重要：ファイル名を変更しないでください
XMPファイルは元の写真と同じファイル名（拡張子のみ .xmp に変更）で生成されます。 XMPと写真のファイル名が一致しないとカラーラベルが反映されません。
  • アップロード前に写真のファイル名を変更しないでください
  • ダウンロードしたXMPのファイル名を変更しないでください
  • 例: DSC_0001.jpg → DSC_0001.xmp のペアで動作します
  • RAWファイル（.ARW, .CR3, .NEF 等）でも同様に動作します
⚠
現像設定の上書きにご注意ください
SELEKTのXMPファイルにはカラーラベル情報のみが含まれています。 すでに現像ソフトで編集済みの写真に対してXMPを読み込むと、 既存の現像設定（露出、ホワイトバランス、トーンカーブなど）が失われる可能性があります。
  • XMPは必ず現像を始める前に写真と同じフォルダに配置してください
  • 既に現像設定がある場合は、XMPの配置前にバックアップを取ってください
  • Lightroom の場合、カタログに現像設定がある状態で「メタデータをファイルから読み込み」を行うと設定が上書きされます

Adobe Lightroom Classic での読み込み
  1. XMPファイルを写真と同じフォルダに配置
  2. Lightroom Classic のライブラリモジュールを開く
  3. メニュー → メタデータ → フォルダからメタデータを読み込む
  4. または、写真を右クリック → メタデータ → ディスクから設定を読み込み
  5. カラーラベルが反映されます ✓
💡
Lightroom のカラーラベル設定
Lightroom のカラーラベルセット名が「Lightroom デフォルト」以外の場合、ラベルが正しく表示されないことがあります。 メタデータ → カラーラベルセット → Lightroom デフォルト を選択してください。
Capture One での読み込み
  1. XMPファイルを写真と同じフォルダに配置
  2. Capture One でフォルダをカタログまたはセッションに追加
  3. 写真を選択 → メニュー イメージ → メタデータを同期...
  4. または右クリック → メタデータを同期
  5. 「XMPから読み込む」を選択 → カラータグが反映されます ✓
Adobe Bridge での読み込み
  1. XMPファイルを写真と同じフォルダに配置
  2. Bridge でフォルダに移動 → 自動的にXMPが読み込まれます
  3. カラーラベルが即座に反映されます ✓

よくある質問
Q: RAWファイル（ARW, CR3, NEF）でも使えますか？
はい。XMPサイドカーファイルはJPEGだけでなくRAWファイルにも対応しています。 元のファイル名が一致していれば拡張子の種類は問いません。
Q: 既存のXMPファイルがある場合は上書きされますか？
はい、同名のXMPファイルが既にある場合は上書きされます。 既存のメタデータ（現像設定等）が消える可能性があるため、 必要に応じてバックアップを取ってから配置してください。
Q: 保留（Yellow）の写真は書き出されますか？
はい。OK/保留/NG すべての判定済み写真のXMPが生成されます。 未判定の写真のみスキップされます。
