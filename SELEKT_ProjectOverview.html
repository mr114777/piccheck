🎯 1. サービス概要

SELEKT（セレクト）は、カメラマンがクライアントへ写真を共有し、
クライアントが直感的に写真をOK / 保留 / NGで振り分けられるWebサービスです。

結婚式・撮影会・企業撮影など、大量の写真からお気に入りを選ぶ作業を
スマートフォンでもストレスなく行えるよう設計されています。

✨ 4. 主な機能

📤

写真アップロード

ドラッグ&ドロップ対応。1セッション最大50枚、1枚25MBまで。サムネイル自動生成。

🗂️

グループ管理

色分けされたグループで写真を分類。AIによる自動グループ分けにも対応。

✓✕△

3段階セレクト

OK / 保留 / NG の3ステータスで直感的に振り分け。進捗バーで完了率を表示。

🔍

AIセンシティブ検出

NSFWJS（ブラウザ内AI）で水着・下着等を自動検出。プライバシー保護のためサーバー送信なし。

📱

モバイル最適化

スマホでフル操作可能。ピンチズーム、スワイプ、ボトムシートUIに対応。

📥

柔軟なダウンロード

フルサイズ / SNS圧縮（1〜3MB）を選択可能。ステータス別にZIPダウンロード。

⏰

有効期限管理

セッションに自動期限設定（デフォルト7日）。期限切れデータは自動削除。

🌙

ダークモードUI

写真が映えるダークテーマ。ノイズテクスチャ対応など、洗練されたデザイン。
